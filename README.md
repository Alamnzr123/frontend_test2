## Running on Local Development

1. Clone Frontend Repository - **https://github.com/Alamnzr123/frontend_test2**
2. Clone Backend Repository - **https://github.com/Alamnzr123/backendtest_2**
3. NPM install
4. Adding Database
5. Run using npm
```
  npm run dev
```
